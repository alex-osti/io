# MakerToo/Io-Natural: Digital Business Ecosystem 2026

A high-performance digital infrastructure for the **Io-Natural** project, featuring dedicated monitoring, automation flow reports, and strategic value analysis.

## 🚀 Overview

This repository contains the static frontend for the **Io-Natural Digital Ecosystem**. It serves as a comprehensive dashboard for tracking infrastructure health, marketing automation, SEO metrics, and overall digital asset valuation.

### Core Modules
- **Pregled stanja (Dashboard)**: Real-time overview of the digital sovereignty and market value.
- **Infrastruktura (Infrastructure)**: Technical breakdown of VPS, Docker, and Redis caching.
- **Avtomatizacija (Automation)**: Reports on CRM integration, CAPI tracking, and conversion funnels.
- **SEO Uspeh**: Semantic SEO frame performance and organic growth indexing.
- **Vrednost (Valuation)**: Strategic asset assessment based on engineering complexity and market impact.

## 🛠 Tech Stack
- **Frontend**: HTML5, Tailwind CSS
- **Design**: Modern Dark Mode Aesthetics, Glassmorphism, Material Symbols
- **Fonts**: Manrope (Headlines), Inter (Body)

## 📁 File Structure
- `index.html`: Main Dashboard (Pregled stanja)
- `summary.html`: Executive Strategic Summary
- `sixth.html`: Infrastructure Analysis
- `seventh.html`: Automation & CRM Reporting
- `forth.html`: Asset Valuation & ROI
- `seo.html`: SEO Performance Metrics
- `makertoo-favicon.png`: Brand assets

## 💻 Local Development
To view the dashboard locally, simply open `index.html` in any modern browser, or use a local server:

```bash
npx serve .
```
